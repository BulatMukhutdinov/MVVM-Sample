# Used technologies and principles
* Clean architecture (MVVM for presentation layer)
* Kotlin DSL for gradle
* Android Architecture Components (Room, Paging, Lifecycle, Navigation)
* RxJava 2
* Databinding
* Tumblr API
* DI (Koin)
* CI (Travis)
* Animated vector drawable
